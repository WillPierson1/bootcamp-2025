HTML Basics Tutorial

This tutorial covers some basic html elements, let's dive in!

Getting Started

HTML stands for Hyper Text Markup Language

What you'll learn

You'll be learning how to create web pages with headings, footers, articles and images!

Basic HTML Structure

HTML Documents are structured with tags. The head tag is used to store metadata, such as the page title or linking stylesheets. The body tag stores the main content of your web page, such as text and images like you're viewing now. The whole document should be surrounded by html tags, and you should include the DOCTYPE at the top of the file.

Code Example

<h1>This is my Heading! </h1>
<p>This is my paragraph!</p>

Text Formatting

Text can be formatted in a few different ways. For example: bold text, Italic text, small text and highlighted text.
